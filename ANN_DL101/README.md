## Getting Started

1. Clone the repo 리파지토리 클론하기
   ```sh
   git clone https://github.com/phdshinai/ANN_DL101.git
   ```
2. Install dependencies 디펜던시 설치하기
   ```sh
   cd ANN_DL101    
   pip install -r requirements.txt
   ```
